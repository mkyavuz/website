# Mustafa Kerem — Personal Website

My personal website and blog, built with plain HTML and CSS and hosted on GitHub Pages.

**Live site:** https://mkyavuz.github.io/website/

## Pages
- **Home** (`index.html`) — intro, about, featured projects, latest posts
- **Projects** (`projects.html`) — full list of projects with links to their repos
- **Blog** (`blog.html`) — writing, with individual posts under `posts/`

## Tech
No frameworks or build step — just static files:
- `index.html`, `projects.html`, `blog.html` — pages
- `posts/` — individual blog posts
- `css/style.css` — styling

## Running locally
Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.
